# Local Trades Chatbot

AI receptionist for local service businesses.

## Goal
Help roofing companies, landscapers, plumbers, HVAC companies, and other trades:

- Answer customer questions 24/7
- Capture qualified leads
- Book appointments
- Reduce missed calls

## Planned Stack

- React frontend
- Node.js + Express API
- PostgreSQL database
- OpenAI API
- Calendar integrations
- SMS notifications

## MVP Features

- Chat widget
- Lead collection
- Appointment requests
- Contractor knowledge base
- Admin dashboard
